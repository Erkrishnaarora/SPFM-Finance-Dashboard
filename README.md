SPFM-Finance-Dashboard

Overview

SPFM Finance Dashboard is a web-based personal finance management application developed using Python and Streamlit. It provides a structured and interactive platform for tracking income and expenses while delivering analytical insights through data visualization.

The application is designed to be simple, efficient, and user-friendly, making it suitable for individuals who want to manage their finances without relying on complex financial software.

Problem Statement

Managing personal finances manually often leads to unorganized records and limited visibility into spending patterns. Many existing tools are either overly complex or not tailored to individual requirements.

This project addresses these challenges by offering a lightweight and interactive dashboard for efficient financial tracking and analysis.

Objectives

- Develop a centralized system for managing personal financial data
- Enable efficient tracking of income and expenses
- Provide clear visual insights into spending behavior
- Simplify financial analysis through an interactive interface

Features

- Add income with customizable sources
- Add expenses with user-defined categories
- View complete financial records in tabular format
- Real-time calculation of total income, expenses, and balance
- Category-wise expense analysis
- Income versus expense comparison through graphical representation
- Data visualization using bar charts and pie charts
- Search functionality for specific transactions
- Option to delete individual records
- Option to clear all stored data
- Automatic alerts when expenses exceed income

Tech Stack

- Programming Language: Python
- Framework: Streamlit
- Data Handling: Pandas
- Visualization: Matplotlib

System Architecture

Frontend

Built using Streamlit to provide an interactive user interface

Backend

Implemented in Python with Pandas for data processing

Data Storage

CSV-based storage system for simplicity and portability

Installation and Setup

1. Install dependencies
   pip install streamlit pandas matplotlib

2. Navigate to the project directory

3. Run the application
   streamlit run student_result_gui.py

4. Open the local server link displayed in the terminal
